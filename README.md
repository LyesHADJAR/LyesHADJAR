**Algerian Dialect Diacritization — Dataset & Modeling**

*ENSIA · Collaboration with Portland State University*

<br/>

Co-leading the construction of the **first Algerian dialect diacritization corpus** —
6,086 sentences, 95.2% character-level inter-annotator agreement,
built via a 3-phase human annotation pipeline over LLM pre-annotation.

Benchmarking BiLSTM-CRF, AraBERT, MARBERT, and DziriBERT under MSA/dialect transfer-learning strategies.
